## 📌 Deskripsi Proyek

Proyek ini merupakan sistem **deteksi fraud transaksi keuangan** berbasis Machine Learning yang menggunakan pendekatan **ensemble learning** untuk meningkatkan performa prediksi.

Sistem ini dirancang untuk menganalisis data transaksi dan mengklasifikasikan apakah suatu transaksi termasuk **fraud (penipuan)** atau **normal**.

Pendekatan yang digunakan:
- 🌳 **XGBoost Classifier** (model utama berbasis boosting)
- 🧠 **Multi-Layer Perceptron (MLP)** (deep learning model)
- 🔗 **Ensemble/Hybrid Model** (kombinasi hasil prediksi)

---

## 🎯 Tujuan Project

- Mendeteksi transaksi fraud secara otomatis
- Meningkatkan akurasi prediksi menggunakan ensemble learning
- Menyediakan sistem backend API untuk integrasi
- Menyediakan dashboard analisis data

## ▶️ Cara Menjalankan Project

### 1. Clone Repository
```bash
git clone https://github.com/username/Portofolio_2.git
cd Portofolio_2
````

---

### 2. Install Dependencies

```bash
pip install -r API/requirements.txt
```

---

### 3. Jalankan Backend API

```bash
cd API
python fraud_api.py
```

📍 Akses API:

```
http://127.0.0.1:5000
```

### 4. Jalankan Dashboard

```bash
cd Deployment

```
